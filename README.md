# FitBuddy AI

## Problem

Many fitness beginners struggle to calculate their BMI, calories, and daily macronutrient requirements.

## Solution

FitBuddy AI is a lightweight fitness planning agent that helps users calculate BMI and provides macro recommendations based on their fitness goal.

## Agent Architecture

User → Fitness Agent → BMI Tool + Macro Tool

## MCP Tools

1. BMI Tool
2. Macro Tool

## Security

No API keys are stored in the code.

## How to Run

```bash
python app.py
```

## Output

The system returns BMI, calories, protein, carbohydrates, and fat recommendations.
